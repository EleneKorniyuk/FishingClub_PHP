# FishingCLUB — Рыболовный комплекс

Веб-приложение для рыболовного комплекса с возможностью просмотра услуг, мероприятий, бронирования и обратной связи. Полноценная админ-панель для управления контентом.

## Стек технологий

*   **PHP 7.4+** (Серверный язык)
*   **MySQL** (База данных)
*   **PDO** (Работа с БД - защита от SQL-инъекций)
*   **HTML5 + CSS3** (Вёрстка)
*   **JavaScript** (Интерактивность)
*   **Session** (Авторизация)
*   **password_hash()** (Безопасное хранение паролей)

## Админ-панель

* **Панель управления**: Просмотр статистики (количество записей).
* **Рыболовные услуги**: Добавление, редактирование, удаление.
* **Услуги комфорта**: Добавление, редактирование, удаление.
* **Мероприятия**: Добавление, редактирование, удаление, поиск.
* **Бронирования**: Просмотр, изменение статуса (новая/подтверждена/отменена).
* **Отзывы**: Просмотр, изменение статуса (новый/в работе/отвечен/закрыт), удаление.

## Особенности реализации

1. **Безопасность**
    * PDO с подготовленными запросами - защита от SQL-инъекций
    * password_hash() и password_verify() - безопасное хранение паролей
    * htmlspecialchars() через функцию e() - защита от XSS
    * Session - авторизация пользователей
    * requireAuth() - проверка авторизации в админке
2. **Адаптивность**
    * Flexbox и Grid для адаптивной вёрстки
    * Медиа-запросы для мобильных устройств
    * Адаптивные карточки услуг и мероприятий
3. **Пагинация** - реализована на странице услуг:
    * ITEMS_PER_PAGE (5 записей на страницу)
    * Номера страниц с активным состоянием
4. **Поиск**: Реализован по назаванию/описанию мероприятия.
5. **Фильтрация**: Реализована по статусу бронирований/отзывов.
6. **Выпадающие списки**: Динамическое обновление списка услуг в зависимости от типа.
7. **Интерактивность**
    * Валидация телефона - оставляет только цифры и знак +
    * Подтверждение удаления - подтверждение перед удалением записи
    * Детали отзывов и бронирований - просмотр полной информации

### 1. Системные требования
* Веб-сервер: **Apache** / **Nginx** / **OpenServer** / **XAMPP** / **MAMP**
* PHP: **7.4** или выше
* MySQL: **5.7** или выше

git clone https://github.com/EleneKorniyuk/FishingClub_PHP.git
